# Zoincas

Zoincas is a modern personal finance management application that helps you track expenses, manage budgets, and gain insights into your financial health. With an intuitive interface and powerful analytics, Zoincas makes it easy to monitor your spending, set financial goals, and understand your money flow through beautiful visualizations.

## ðŸš€ Features

- **Modern Stack**: Built with Next.js 15, React 19, Hono.js, and TypeScript
- **Styling**: Modern styling with Tailwind CSS
- **UI Components**: Beautiful, accessible components using Radix UI and shadcn/ui
- **State Management**: Efficient state management with Zustand
- **Data Fetching**: Optimized data fetching with TanStack Query
- **Form Handling**: Robust form management with React Hook Form and Zod validation
- **Database**: Serverless PostgreSQL with Neon DB and Drizzle ORM for type-safe database operations
- **Authentication**: Secure authentication powered by Better Auth with Google OAuth
- **AI Integration**: Powered by AI SDK with Google's Gemini for intelligent features
